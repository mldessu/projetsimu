Calcul du volume d'une $N$-sphre (20 points)

Introduction

Une $N$-sphre de rayon $R$ est dfinie comme l'ensemble des points situs une distance infrieure ou gale $R$ de l'origine dans un espace $N$ dimensions. Si on utilise les coordonnes cartsiennes $(x_1, x_2, \dots, x_N)$ pour dcrire un point, alors ce point est situ l'intrieur de la $N$-sphre si le critre suivant est respect :

$$ \sum_{i=1}^{N} x_i^2 \leq R^2, \quad N=1,2,\dots $$

Voici quelques exemples de $N$-sphres afin de clarifier ce concept :

  • Une 3-sphre est la sphre habituelle en trois dimensions. Son volume est la boule dont la sphre est le contour.
  • Une 2-sphre est un cercle. Son volume est l'aire du disque circonscrit par le cercle.
  • Une 1-sphre est forme de deux points sur une droite qui sont situs gale distance de l'origine. Son volume est dfini comme la longueur du segment.

Ce concept peut sembler abstrait, mais il apparat naturellement en physique statistique quand on travaille dans l'espace de phase pour tudier les proprits d'un ensemble de $N$ particules avec une nergie fixe (ex. : gaz idal dans une bote ferme).

Dans ce mini-devoir, on vous demande de calculer le volume d'une $N$-sphre de rayon unitaire $R=1$ l'aide de la mthode de Monte-Carlo vue en classe pour le calcul d'intgrales. Le rsultat est dj connu, savoir :

$$ V_N = \frac{\pi^{N/2}}{\Gamma\left(\frac{N}{2} + 1\right)}, $$ o $\Gamma(x)$ est la fonction gamma, ce qui vous permettra de calculer l'erreur afin d'analyser le comportement de l'algorithme.

Rponse a):

Les mthodes de Monte-Carlo sont avantageuses pour lintgration en haute dimension car leur erreur moyenne dcrot proportionnellement $ \frac{1}{\sqrt{N_{\text{tot}}}} $, indpendamment du nombre de dimensions $N$, et elles ncessitent ainsi beaucoup moins de points dvaluation que les mthodes de quadrature, dont le cot crot exponentiellement avec $N$. Cette efficacit et robustesse, indpendantes de la forme de la fonction (mme impossible rsoudre analytiquement), en font souvent la seule option pratique pour des intgrales de grandes dimensions.

import numpy as np
import math
import matplotlib.pyplot as plt

# Paramtres gomtriques
R = 1.0 # Rayon de la sphre (m)

# Paramtres gnraux de simulation
D_val = [3,6]  # Nombre de dimensions
Ntot_val = 100 * 2 ** (np.arange(0, 5, 1))  # Nombre de points par essai
Ness = 100 # Nombre d'essais par simulation

# Boucle sur le nombre de simulations
ND = len(D_val)
NNtot = len(Ntot_val)

V = np.zeros((ND, NNtot))  # Volumes calculs pour chaque srie d'essais
inc = np.zeros((ND, NNtot)) # Incertitudes pour chaque srie d'essais
err_arr = np.zeros_like(V)

for d in range(0, ND):
    D = D_val[d]  # Dimension
    Vtot = ( R * 2 ) ** D # Volume du domaine
    Vth = (np.pi ** (D / 2)) / math.gamma(D / 2 + 1)
   
    for n in range(0, NNtot):
        Ntot = Ntot_val[n]  # Nombre de points

        Vind = np.zeros(Ness) # Volumes calculs pour chaque essai individuel
        e = np.zeros_like(Vind)
        for k in range(0, Ness): # Boucle sur les essais
            # Gnration des nombres alatoires (distribution uniforme)
            np.random.seed() # Initialise le gnrateur de nombres pseudo-alatoires afin de ne pas toujours produire la mme squence  l'ouverture de Python...
            pts = R * np.random.uniform(low=-R, high=R, size=(Ntot, D)) # Coordonnes des points
            
            # Calcul du volume
            distances = np.linalg.norm(pts, axis=1)
            Nint = np.sum(distances <= R) # Nombre de points  l'intrieur
            Vind[k] = Nint / Ntot * Vtot # Volume calcul pour cet essai
            e[k] = np.abs((Vind[k] - Vth)/ Vth) # Erreur pour cet essai

        err_arr[d,n]= np.mean(e) # Erreur relative sur tous les essais
        V[d, n] = np.mean(Vind) # Volume moyenn sur l'ensemble des essais
        inc[d, n] = np.std(Vind) # Incertitude sur l'ensemble des essais (cart-type)

inc_rel = inc / V * 100 # Calcul de l'incertitude relative
print("Volumes:", V)
print("Incertitude relative:", inc_rel)
print("Erreur relative:", err_arr)
